GriffinID program

This a a service program to be used with the griffin WSPR/QRSS beacon board. This board transmits independent of a computer so the transmited beacon string must be loaded into the the FPGA. This program takes your string input and formats it for inclusion in the firmware of the FPGA.

This is program will generate:

  1. WSPR beacon string in encoded format
  2. QRSS beacon string in encoded in morse and output in a hex string format

This program follows the restrictions of the WSPR software in the call signs are limited to 6 characters abd the third character must be a number and padded with blank before and after the number. The location can be entered either as a 4 character Maidenhead Grid or Latitude/Longitude but limited to 2o by 1o cells. You will also need to enter the dBm of the signal transmit strenght. To get the dbm to Watts table follow the link.

If you generating a QRSS string you only need to enter your call sign
