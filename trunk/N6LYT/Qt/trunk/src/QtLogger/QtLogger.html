QtLogger program

QtLogger is a Amateur Radio logging program to log radio contacts. The prgram is designed to work with the ghpsdr3 program s suite written by John, Melton G0ORX/N6LYT. The logger should pickup the frequency and Mode from a running dspserver.

Features

Radio data tracking
QtLogger listens for a UDP broadcast of the radio frewquency and mode and set these feilds to teh current values.
Read and write XML file
The base file format is XML which is in a format similar to the common ADIF files used by other logging programs. The XML file comes with an XSLT file that allows for viewing of the log file witha web browser.
Read and write ADIF file
The program can read and write ADIF file formats for compatibility with other logging programs or LOTW.
Read categorical data an XML file
Categorical data is stored in XML such as countries and subdivisions. these are used to auto complete these fields.
Date and Time auto input using the system clock
the add contact dialog tracts the current data and time of the computer system clock and record them when the contact is stored.
Callsigns must conform to standard Amateur Protocal
All callsign are checked against a validator for a correctly formed callsign. The / place suffixes are allowed in this check.
Logical tabbing sequence and autocompletion
