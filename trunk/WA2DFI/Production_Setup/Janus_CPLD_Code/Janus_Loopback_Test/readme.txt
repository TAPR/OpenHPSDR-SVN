Janus_Loopback_Test directory

<Janus_wrap.pof> is the binary programming image for the Janus loopback test CPLD code.
<Janus.pof> is the binary programming image for the Janus production CPLD code that was programmed into all shipped Janus boards.

<Janus_wrap.v> is the source code for the Janus loopback test CPLD code.
