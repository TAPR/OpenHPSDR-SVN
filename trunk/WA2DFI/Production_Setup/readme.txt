Production_Setup directory


<Docs> contains documentation on the test setup and procedures
<Janus_CPLD_Code> contains the operating and test versions of the Janus CPLD code (binaries and source)
<Ozy_FPGA_Code> contains the test versions of the Ozy FPGA code (binaries and source) (the operating version is loaded by the FX2 at run time)
<Test_Fixture_Hardware> contains schematics of the loopback hardware that you will need to production test Ozy and Janus
<Working_Directory> contains all of the batch files, executables and CPLD/FPGA images that you will need to run the test suite
